# 功能特点：

* 支持多种方式创建 Bytes
* 支持字节数组、ByteBuffer 的操作
* 支持 Transformer: 内置 copy、contact、reverse、xor、and、or、not
* 支持 Hash: 内置 md5、sha1、sha256 
* 支持 mmap 的操作
